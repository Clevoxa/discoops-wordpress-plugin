# Discoops WordPress Plugin

Official WordPress plugin for Discoops.

This plugin connects your WordPress site to the Discoops platform so you can execute Google Discover optimization actions faster, streamline editorial workflows, and act directly from WordPress.

## Requirements

- A WordPress site
- An active Discoops account
- Access to the Discoops platform

# Extension WordPress Discoops

Extension WordPress officielle de Discoops.

Cette extension connecte votre site WordPress à la plateforme Discoops afin d’exécuter plus rapidement vos actions d’optimisation Discover, fluidifier vos workflows éditoriaux et agir directement depuis WordPress.

## Prérequis

- Un site WordPress
- Un compte Discoops actif
- Un accès à la plateforme Discoops
